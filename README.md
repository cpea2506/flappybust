## How to build and run

### Prerequisite

- Rust >= 1.67
- [Cargo Watch](https://crates.io/crates/cargo-watch) (optional)

### Steps

1. Clone the project
2. Run the command:

```bash
$ cargo run

# or
# require cargo-watch
# this will watch for file changes and
# concurrently run Bevy with dynamic features

$ cargo watch_dynamic
```
